# 🏋️‍♂️ App Contador de Repetições

Um app simples e responsivo para contar repetições por série na academia. Desenvolvido com **HTML5**, **CSS3**, **JavaScript** e usando **Vite** para um fluxo de desenvolvimento rápido.
